template<typename T>
struct boost::dispatch::meta::model_of< T >

Model generator meta-function.

Build, for any given type T, a Metafunction which can reconstruct a new type U similar to T but by using a different underlying type.

Models:

Metafunction

Semantic:

For any type T,

meta::model_of<T>::type;

defines a template alias so that:

std::is_same<meta::model_of<T>::type<meta::value_of_t<T>>, T>::value

evaluates to true.

Extension Point:

meta::model_of can be specialized for user-defined types by either:

  • Directly overloading the meta::model_of class for a given type
  • Specialize (eventually through SFINAE) the ext::model_of class

Specialization for meta::model_of are provided for most of standard and Boost types. See the External adapters page.
