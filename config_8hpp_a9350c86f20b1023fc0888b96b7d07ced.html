#define BOOST_DISPATCH_RESTRICT

Mark a pointer or reference as restricted.

This macro expands to a potential compiler specific parameters attribute indicating that a given pointer or reference is not aliasing any other pointer or reference.

Usage:
1 void* BOOST_DISPATCH_RESTRICT ptr = malloc(1337);
