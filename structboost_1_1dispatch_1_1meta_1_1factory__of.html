template<typename T, typename Limit = typename meta::primitive_of<T>::type>
struct boost::dispatch::meta::factory_of< T, Limit >

Recursive meta-model generator.

For any type T, returns a Lambda Expression able to reconstruct T using a different primitive type.

Template Parameters
TType to build a type-factory of
Limit
Models:

Metafunction

Semantic:

For any type T:

using type = boost::dispatch::meta::factory_of<T>::type;

return a template alias so that, for any type U:

using X = type<U>;

generates a type X so that:

std::is_same<boost::dispatch::primitive_of_t<X>, U>::value

and for which the boost::dispatch::meta::model_of template aliases are equivalent.

Put in another way, boost::dispatch::meta::factory_of is a recursive application of boost::dispatch::meta::model_of so that every nested type are traversed during reconstruction.
