template<class T, class Enable = void>
struct boost::dispatch::ext::value_of_cv< T, Enable >

SFINAE-enabled extension point for meta::value_of with qualifiers.

This extension point allows customization of the meta::value_of Metafunction by letting user defined SFINAE-enabled context in which they can specialize meta::value_of for family of types with cv-qualifiers
